# ThoughtQueue â€“ Deferred Decision Management System

A console-based Core Java application that helps users organize thoughts and decisions using the Queue (FIFO) principle.


## Features

- Add Thought
- View All Thoughts
- View Next Thought (Peek)
- Complete/Delete Thought (Dequeue)
- Save & Load Thoughts using File Handling

## Why This Project?

People often act impulsively or forget important ideas.  
This project provides a structured way to store and process thoughts using the Queue (FIFO) data structure so that older thoughts are reviewed first.

## Concepts Used

- Object-Oriented Programming (OOP)
- Queue (FIFO)
- ArrayDeque / Queue Interface
- File Handling
- Exception Handling
- Menu-driven architecture
- Scanner class for input
